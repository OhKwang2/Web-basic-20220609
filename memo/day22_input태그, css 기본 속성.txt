@입력 양식 작성하기

	#<input> 태그
	* 키보드 입력 창
	- text : 일반 문자 입력
	- password : 비밀번호 입력
	- search : 검색할 때 입력하는 필드 (끝에 x 표 생성)
	- url : 주소를 입력
	- email : 이메일 주소 입력
	- tel : 전화번호 입력
	* 
 	- checkbox : 2개 이상 체크
	- radio : 1개만 체크
	- number : 스핀박스 생성
	- range : 슬라이드 막대 생성
	- date, month, week, time, datetime, dateime-local : 시간 관련
	- file : 파일을 첨부할 수 있는 버튼 생성
	- hidden : 사용자에게는 보이지 않지만, 서버로 넘겨주는 값이 있는 필드 생성

	#<input> 태그 속성
	- autofocus : 페이지 열리면 해당 input 태그 포커싱
	- placeholder : 미리 입력된 값 생성
	- readonly : 입력값 수정 불가 (vlaue로 값 입력)
	- required : 필수입력 항목

	#<textarea>
	- 텍스트 입력 공간 생성

	#<select>
	- optgroup, option에 있는 값 선택
	#<datalist>
	- option에 있는 값을 선택하거나 값 입력시 바로 유사한 값을 추천

	#<pre>
	- 해당 태그 내 값에서 줄바꿈시 자동 적용되서 출력

@CSS 기본

	#스타일 형식
	- 브라우저 기본 스타일 : 브라우저 자체의 기본 스타일
	- 사용자 스타일(인라인 스타일, 내부 스타일 시트, 외부 스타일시트)로 구분
	
	#캐스케이딩
	- 위에서 아래로 흐른다는 뜻, 즉 계딴식으로 적용된다는 의미
	- 스타일 우선순위 : 우선순위에 따라 위에서 아래로 스타일 적용
	- 스타일 상속 : 태그들의 포함 관계에 따라 부모 요소의 스타일을 자식 요소로, 위에서 아래로 전달
	- css는 외부 스타일시트로 적용하는게 일반적
	- javascript도 마찬가지

	#선택자
	- 전체 선택자 : *{스타일 규칙} 전체 스타일, 초기 세팅을 잡기위해 사용
	- 타입 선택자 : 태그명{스타일 규칙}
	- class 선택자 : 문서 안에 여러 번 반복할 스타일이라면 class 선택자로 정의
	- id 선택자 : 문서 안에 한번만 사용한다면  id선택자로 정의
	-그룹 선택자 : 여러개의 선택자를 ,로 구분하여 한 번에 정의

	#선택자 우선순위
	- !important : 어떤 스타일보다 우선시 되는 스타일
	  (스타일 규칙 마지막에 !important 입력)
	- 인라인 스타일 : 해당 태그에만 적용되는 스타일
	- id 스타일 : id에 적용되는 스타일
	- class 스타일 : class에 적용되는 스타일
	- 타입 스타일 : 태그 자체에 적용되어 있는 스타일
	
	#스타일 상속
	- 스타일의 모든 속성이 부모 요소에서 자식 요소로 상속되는 것은 아님
	(배경색은 상속되지 않음)
