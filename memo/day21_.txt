@웹 개발환경 만들기

	#웹 브라우저
	- 크롬 브라우저 사용
	#웹 편집기
	- vs code 

	#VS CODE 설치, 세팅
	- VS 코드 설치 후 폴더 세팅
	- 확장 프로그램의 korean, liveserver 설치
	- 파일 -> 기본설정 -> 설정 -> Font Family (D2Coding)
	- 파일 -> 기본설정 -> 설정 -> Live Server Config -> 크롬 프라우저를 기본으로 설정

	#HTML
	- Hyper Text MarkUp Language
	- HTML - 문서의 기본 구조
	- <html><head></head><body></body></html>

	#DOCTYPE html
	- 해당 파일의 타입을 지정하는 태그

	#<html>
	- 속성을 가지는 문서
	- lang에 다른 언어 코드 사용하면 언어번역 버튼이 자동 생성

	#<meta> 태그 : 문자형 등 문서정보를 제공하는 태그

	#<body> 태그 : 문서의 내용이 들어가는 태그
	- 일반적으로 header, main, footer 태그 영역으로 구성

	#semantic 태그
	- 이름만 봐도 의미를 알 수 있는 HTML 태그
	- 왜 사용할까? 웹 사이트를 검색할 때 필요한 내용을 정확히 찾을 수 있다.
	- 본문내용 외에 사이드 바 영역을 나타내는 aside 등

@다양한 내용 입력하기

	#<h>태그
	- 제목을 표시하는 태그
	- h1~h6까지 있음

	#<br>
	- 줄바꿈

	#<p>
	- 단락

	#<b>
	- 글자 굵게

	#<i>
	- 이테릭체로 표시

	#<ol>, <li>
	- 리스트 목록 나타낼 때 사용
	- 각 항목 앞에 순서가 붙게 됨.

	#<ul>, <li>
	- 순서가 없는 리스트 나타낼 때 사용

	#<dl>, <dt>, <dd> - 잘 안씀
	- <dt> : 이름(제목)
	- <dt> : 값(설명)
	- 하나의 <dt>에 여러개의 <dd> 값을 가짐.

	#<table>
	<tr>
	     <th></th>
	     <td></td>
	</tr>

	#<a>
	- 다른 문서 혹은 다른 사이트로 바로 연결해주는 기능
	- target = "_blank" <- 연결된 문서가 새 탭으로 열림

@입력 양식 작성하기

	#★<form>
	- ★method :
		get : 256~4,096byte까지만 서버 넘길 수 있음
		post : 입력한 데이터의 길이 제한 X, 입력한 내용도 드러나지 않음.
	- name
	- ★action : 
		매핑 주소를 입력하는 태그
	- target
	
	#<lable>
	- <input> 태그와 같은 폼 요소에 레이블을 붙일 때 사용
	- text, password, search, url, email, tel, checkbox, raio, number, date, file, hidden 주로 사용

	#<input>
	- type : serch(전체 지우는 속성), url(해당 링크로 이동), email(@안 들어가면 안 되는 형식), tel(전화번호 입력 필드), radio(여러 항목중 하나만 선택), checkbox(여러개 선택)
