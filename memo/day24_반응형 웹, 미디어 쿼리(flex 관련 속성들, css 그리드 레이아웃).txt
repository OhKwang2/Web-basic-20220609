	- 전체적인 디자인에 대한 개념
	- 웬만한 레이아웃은 flex로 잡음

	 #플렉스 박스 레이아웃
	- 그리드 레이아웃을 기본으로 플렉스 박스를 원하는 위치에 배치하는 것
	- 여유 공간에 따라 너비나 높이, 위치를 자유롭게 변형할 수 있음.
	-float은 마진 위에 붕 뛰우는 개념
	 (블록 위로 영역이 올라올 수 있음)
	- float는 개별 자식 영역에 주는 것
	- ★flex는 부모의 위치를 바꾸는 것

	#주축과 교차축
	- flex를 사용하지 않을 경우 주축은 아래쪽방향이지만, flex를 사용할 경우 주축이 우측으로 바뀌게 됨.
	- 교차축은 주축 끝나는 지점으로 넘어가면 자동으로 다음줄로 넘어감
	- flex-wrap 설정으로 자동으로 영역이 넘어가면 줄을 바꿔줌
	- flex-wrap을 설정하지 않을경우 contents 사이즈가 축소됨.

	- 주축방향을 justify
	- 교차축 방향을 align
	- flex-direction : 축 방향을 바꿔줄 수 있음
	- row : 기본 설정 (좌에서 우)
	- row-revers : 우에서 좌
	- column : 위에서 아래 방향
	- column-revers : 아래에서 위\

	#flex-wrap
	nowrap : 플레스 항목을 한 줄에 표시합니다.
	wrap : 플렉스 항목을 여러 줄에 표시합니다.
	
	#justify-content
	- 플렉스 항목을 주축 방향으로 배치할 때 배치기준
	- flex-start : 주축의 시작지점에 맞춰 배치
	- flex-end : 주축의 끝점에 맞춰 배치합니다.	
	- center : 주축의 중앙에 맞춰 배치합니다.
	- space-between : r첫 번째 항목과 끝 항목을 주축의 시작점과 끝점에 배치한 후 나머지 항목은 그 사이에 같은 간격으로 배치합니다.
	- space-around : 모든 항목을 주축에 같은 간격으로 배치합니다.

	 #align-item, align-self
	- 교차축을 기준으로 배치하는 방법 조절
	- 교차축에서 특정 항목만 지정하고 싶다면 align-self 속성 사용
	- flex-start : 교차축의 시작접에 맞춰 배치합니다.
	-flext-end : 교차축의 끝점에 맞춰 배치합니다.
	- center : 교차축의 중앙에 배치합니다.
	- baseline : 교차축의 문자 기준선에 맞춰 배치합니다.
	- stretch : 플렉스 항목을 늘려 교차축에 가능 차게 배치합니다.

	#align-content
	- 플렉스 항목이 여러줄로 표시될 때, 교차축 기준의 배치 방법 지정 (교차축 내 row가 여러개일 경우 사용)
	- flex-start : 교차축의 시작지점에 맞춰 배치합니다.
	- flex-end : 교차축의 끝점에 맞춰 배치합니다.
	- center : 교차축의 중앙에 맞춰 배치합니다.
	- space-between : 양교차축에 끝점을 맞추고 나머지 항목을 그 사이에 배치
	- space-around : 여백 동일하게 끝점 사이에 배치
